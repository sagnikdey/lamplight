Components

Reusable, token-driven UI building blocks exported from @lamplight/ui. Every component maps to Figma variants and supports light and dark themes automatically.

Install and import

Lamplight is a monorepo package. Import components directly — tokens and styles are wired through the app layout.

import { Button, Badge, Input } from '@lamplight/ui';

// Tokens are loaded in globals.css
// Theme switching via next-themes + data-theme

Built with

This design system demonstrates end-to-end design engineering — from Figma variables to production React components.
