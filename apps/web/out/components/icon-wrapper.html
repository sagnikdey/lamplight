Icon wrapper

Consistent icon sizing and semantics for Lucide icons across the system.

Atoms

Import

import { IconWrapper } from '@lamplight/ui';

When to use

Wrap Lucide icons in IconWrapper for consistent sizing across the system. Prefer the size prop over CSS width/height on SVGs.

Examples

Usage

import { Search } from 'lucide-react';
import { IconWrapper } from '@lamplight/ui';

<IconWrapper
  icon={<Search aria-hidden />}
  size="sm"
  className="text-[color:var(--color-icon-action)]"
/>

Props

PropTypeDefaultDescription
icon*ReactNode—Lucide icon element
size'sm' | 'md' | 'lg''md'Constrained icon box size
classNamestring—Token-based colour classes for icon fill

Accessibility

Accessibility notes

Decorative icons should include aria-hidden. Interactive icon-only controls need a visible label or aria-label on the parent button or link.
