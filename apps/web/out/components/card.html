Card

Composable card with shadow variants and header, media, body, and footer slots.

MoleculesFigma: 136:946

Import

import { Card, CardHeader, CardBody, CardFooter } from '@lamplight/ui';

When to use

Use cards to group related content — articles, settings panels, or dashboard widgets. Compose with sub-components for production layouts; use boolean props for quick Figma-parity demos.

Anatomy

Card container with optional CardHeader, CardMedia, CardBody, and CardFooter slots. Shadow axis: raised, overlay, or none.

Examples

Usage

<Card shadow="raised">
  <CardMedia src="/image.jpg" alt="Cover" />
  <CardBody title="Article title" subtitle="Summary text" />
  <CardFooter direction="horizontal">
    <Button variant="outline" size="sm">Read more</Button>
  </CardFooter>
</Card>

Props

PropTypeDefaultDescription
shadow'raised' | 'overlay' | 'none''raised'Elevation and surface token pairing
disabledbooleanfalseMuted, non-interactive card state
headerboolean—Render demo header slot
imageboolean—Render demo media slot
bodyboolean—Render demo body slot
footerboolean—Render demo footer actions

Accessibility

Accessibility notes

Cards are semantic article elements when used for self-contained content. Images require meaningful alt text via CardMedia. Disabled cards set aria-disabled and reduce opacity.
