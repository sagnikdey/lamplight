Divider

Horizontal, vertical, and labelled dividers for content separation.

Atoms

Import

import { Divider } from '@lamplight/ui';

When to use

Use dividers to separate related content groups — between form sections, list items, or toolbar clusters. Labelled dividers work well for auth flows ('or continue with').

Examples

Usage

<Divider orientation="horizontal" />
<Divider label="or" />
<Divider orientation="vertical" />

Props

PropTypeDefaultDescription
orientation'horizontal' | 'vertical''horizontal'Divider direction
labelstring—Optional centred label text

Accessibility

Accessibility notes

Horizontal dividers use role="separator" with aria-orientation. Labelled dividers expose the label as visible text — not a replacement for headings.
