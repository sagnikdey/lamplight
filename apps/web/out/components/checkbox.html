Checkbox

Single and indeterminate checkboxes with label positioning and disabled states.

Atoms

Import

import { Checkbox, IndeterminateCheckbox } from '@lamplight/ui';

When to use

Use checkboxes for multi-select options or boolean consent. Use IndeterminateCheckbox for select-all patterns in lists and tables.

Anatomy

Radix Checkbox primitive, custom check icon, label with optional left/right positioning.

Examples

Usage

<Checkbox label="Accept terms" />

<IndeterminateCheckbox
  checked={allChecked ? true : noneChecked ? false : 'indeterminate'}
  onCheckedChange={handleSelectAll}
  label="Select all"
/>

Props

PropTypeDefaultDescription
labelstring—Visible label text
labelDirection'right' | 'left''right'Label position relative to the control
checkedboolean | "indeterminate"—Controlled checked state
defaultCheckedboolean—Uncontrolled initial state
disabledbooleanfalseDisables interaction
onCheckedChange(checked: boolean | "indeterminate") => void—Change handler

Accessibility

Accessibility notes

Built on @radix-ui/react-checkbox with keyboard support and aria-checked including indeterminate state. Labels are clickable and associated with the control.
